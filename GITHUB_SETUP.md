# 5-Minute GitHub Pages Setup (No Git, No Command Line)

## Step 1 — Create a new repo
1. Go to https://github.com/new
2. Name: `m3-dashboard` · Public · Create

## Step 2 — Upload the 3 files
Click **uploading an existing file**, drag `index.html`, `Plant_Input_GulfGuard.xlsx`, `README.md` — commit.

## Step 3 — Enable GitHub Pages
Settings → Pages → Source: Deploy from a branch → main / root → Save.

Your URL: `https://<your-username>.github.io/m3-dashboard/`

Share that URL. Anyone with the link opens the dashboard in their browser.

## Updating
When Robert sends a new `index.html`: open the file in the repo → pencil ✏️ → paste → Commit.
GitHub Pages auto-redeploys in ~30 seconds.
